# IsJohnDead

Turtle WoW addon that monitors Johnhealrman's health status in raids and parties.

## Features

- Automatically detects when Johnhealrman joins raid/party
- Monitors health status in real-time
- Sends warning when health drops below 20%
- Announces funny death messages to raid chat
- Configurable output and debug modes

## Commands

- `/ijd status` - Show monitoring status
- `/ijd debug` - Toggle debug mode
- `/ijd output` - Toggle chat output on/off
- `/ijd scan` - Manual raid scan
- `/ijd test` - Test health check

## Installation

1. Download or clone this repository
2. Copy IsJohnDead folder to `Interface/AddOns/`
3. Restart WoW or `/reload`

## Requirements

- Turtle WoW client (Interface 11200)
- Works in both raids and parties
